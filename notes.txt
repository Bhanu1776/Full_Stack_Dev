* Steps to develop a website from scratch with the help of pug

1) npm init -- in project folder
2) install express and pug 
    sudo npm install -g express --unsafe-perm=true --allow-root
    sudo npm install -g pug --unsafe-perm=true --allow-root
3) Create static and views folder in project directory ..
    Views ==> All pug templates 
    Static ==> All css, jss and image files ..
4) Import all required modules in header and and don't forget to include const app = express();
5) Next thing to do is, Add Express static file serving code 
6) And for setting template engine as pug add pug view engine code
7) To start the server set specific port number and add respective code for that ..
8) Create a root(Home)(/) Endpoint
